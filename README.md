# Blogr Landing Page

##  Project Overview

**Blogr Landing Page** is a responsive, static frontend project that simulates the landing page of a modern publishing platform.  
The main objective of this project is to demonstrate **semantic HTML5 structure**, **modern CSS layout techniques**, **interactive navigation without JavaScript**, and **responsive design principles**.

This project is purely frontend-focused and does not include backend functionality or dynamic data handling.

---

##  Purpose of the Project

The purpose of this project is to:

- Practice and apply **semantic HTML5 elements**
- Build a **clean and accessible navigation system**
- Implement **CSS-only dropdown menus**
- Use **Flexbox and CSS Grid** for layout management
- Create a fully **responsive design** for mobile, tablet, and desktop
- Apply **microinteractions and transitions** to improve user experience
- Follow **accessibility best practices**

This project is intended for **educational and portfolio purposes**.
